	VisualGraphAl:
	-	Classes: 	.cpp .hpp 
	-	Resource: 	.png (background)
	-	proj.win32: 	VisualStudio project
	-	proj.ios_mac:	Xcode project
	
	+	cocos2d:	engine, because of (>1Gb) so it's not included. You can install the engine and copy later if you wanna build.
